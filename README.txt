孤雨的话：
我会想继续开发这个N年以前的系统主要原因其实就是想帮助站长快速迁移站点，VPS能够用Wget指令是没错，但是如果站长是购买虚拟主机的话那就难了，下载了再上传你说多麻烦啊！不如直接服务器对传，更省事，所以有了这一系列的增强功能加入。
这个管理系统有多少漏洞说真的我不知道，我算是个php的菜鸟，不太懂得写php，很多功能都是上网找了再略加修改让它更人性化罢了，所以不建议站长将这个系统长期至于虚拟主机下，最好用完直接删除！
不尽善还请见谅！
